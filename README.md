# AI E-commerce Chatbot

## Overview
This project is an AI-powered chatbot built using Python. It helps users interact with an e-commerce platform by browsing products, checking availability, and guiding them through the purchasing process. The chatbot uses natural language processing to understand user queries and provide accurate responses.

## Features
- Browse products by category or keyword
- Check product availability and prices
- Get recommendations based on user input
- Simulated checkout guidance
- Easy integration for any e-commerce platform

## Technologies Used
- Python
- Natural Language Processing (NLP)
- JSON for data handling
